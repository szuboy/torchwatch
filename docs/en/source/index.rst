Welcome to torchwatch's documentation!
=========================================

This library is complementary to the `PyTorch <https://pytorch.org/>`_ project.
PyTorch is an open source machine learning framework.

The `torchwatch <https://github.com/szuboy/torchwatch>`_ package consists of popular basic block for deep learning,
which strives to be an watch for torch, like iwatch for iphone.

Support
----------

You can ask questions in `Github issues <https://github.com/szuboy/torchwatch/issues>`_.
Please make sure you have read this document before asking questions.

If you think this document is helpful to your research,
welcome to click stars to the `Github <https://github.com/szuboy/torchwatch>`_ of this document.
This will encourage author to further improve the content of the document and package.
